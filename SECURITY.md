# Security Policy

## Supported Versions

The following versions of SPS SECURITY are currently supported:

| Version | Supported |
| ------- | --------- |
| 2.x     | ✅ |
| 1.x     | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

### How to report

Send details to:

samuelpontes.security@gmail.com

Include:

- Description of vulnerability
- Steps to reproduce
- Possible impact
- Suggested fix (if any)

### What to expect

- Response within 48 hours
- Fix in next patch release
- Credit in changelog

## Scope

This policy covers:

- Heuristic engine
- Cloud scanner
- Quarantine system
- Real-time monitor
- CLI interface

## Out of scope

- Third-party dependencies
- Python runtime vulnerabilities
- OS-level issues

## Security best practices

Users should:

- Keep Python updated
- Avoid running as root
- Use virtual environment
- Do not scan system-critical folders
